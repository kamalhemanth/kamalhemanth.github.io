Projects

Face and Retina Detection

The Face and Retina Detection project uses Python and OpenCV to implement real-time face and Retina recognition through a webcam feed. Leveraging Haar Cascade Classifiers, the system detects faces and eyes in images or live video streams, marking them with bounding boxes. This project demonstrates computer vision techniques, including image pre-processing, feature extraction, and real-time object detection. It is an ideal example of applying Python for facial recognition applications, showcasing efficiency and accuracy in detecting facial features in various environments.

Library Management

The Library Management System project, developed using Java, is designed to streamline the operations of a library by automating tasks such as book lending, returning, and inventory management. The system allows users to search for books, borrow and return items, and track due dates. Administrators can add, update, or remove books from the database and view user activity. Built with a simple graphical user interface (GUI), the system utilizes Java’s core libraries for database management and file handling, ensuring efficient and reliable management of library resources.

Covid19 Analysis

The COVID-19 Analysis project, developed using Python, focuses on gathering, analyzing, and visualizing real-time data related to the COVID-19 pandemic. The project pulls data from multiple sources, including APIs, to track global and regional trends, including infection rates, recoveries, and death statistics. The system provides insightful visualizations such as graphs and charts, helping users to better understand the spread and impact of the virus.
